Responsive modal full width

Responsive modal full width transition effect

Адаптивное модальное окно раскрывающееся на всю ширину экрана с оригинальным transition эффектом появления и исчезновения.

Можно использовать на сайте с подключенным плагином Bootstrap, а так же без него.

Возможно помещение в контент модали дополнительных элементов, в стилях потребуется дополнительно указать ширину для них, по умолчанию заданы параметры для тегов <p> и <img>
